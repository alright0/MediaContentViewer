# MediaContentViewer

Небольшой скрипт для циклического показа изображений указанного формата(по умолчанию jpg) из папки content в браузере. Для возможности просмотра изображения должны начинаться с 1.jpg

Дерево:
.  
├── content  
│   ├── 0.jpg  
│   ├── 1.jpg  
│   ├── 2.jpg  
│   └── 3.jpg  
├── index.html  
├── script.js  
└── style.css
